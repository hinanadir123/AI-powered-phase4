# Generate Phase 5 Components

Generate Phase 5 task management system components including recurring tasks, search, priorities, tags, and Dapr integration.

## Usage
```
/phase5-generate
```

## What it does
Uses the phase5-task-generator agent to create:
1. Backend API endpoints for recurring tasks and search
2. Frontend React/Next.js components (PriorityDropdown, TagChips, SearchBar, SortSelect)
3. Dapr Pub/Sub integration via HTTP API
4. Comprehensive test cases
5. Documentation

## Example
```
/phase5-generate
```
