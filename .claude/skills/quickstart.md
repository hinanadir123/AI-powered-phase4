# Quick Start

Quick start guide for the Todo AI Chatbot project.

## Usage
```
/quickstart
```

## What it does
Provides an interactive quick start experience:
1. Checks prerequisites (Docker, kubectl, Helm, Minikube)
2. Clones/updates repository
3. Sets up local development environment
4. Builds Docker images
5. Deploys to local Kubernetes
6. Opens application in browser
7. Shows next steps and useful commands

## Example
```
/quickstart
```
