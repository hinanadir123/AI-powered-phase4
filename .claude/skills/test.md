# Run Tests

Run the test suite for the Todo AI Chatbot application.

## Usage
```
/test [component]
```

## Arguments
- `component` (optional): Specific component to test (backend/frontend/all). Default: all

## What it does
1. Runs unit tests
2. Runs integration tests
3. Generates test coverage report
4. Shows test results summary

## Example
```
/test
/test backend
/test frontend
```
