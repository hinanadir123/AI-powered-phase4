# Test Phase 5 Deployment

Validate Phase 5 deployment and advanced features including Kafka/Dapr integration.

## Usage
```
/phase5-test
```

## What it does
Uses the phase5-deployment-tester agent to validate:
1. Cloud cluster deployment (AKS/GKE)
2. Kafka/Dapr integration
3. Recurring tasks functionality
4. Reminders, priorities, tags
5. Search, filter, and sort features
6. CI/CD pipeline
7. End-to-end workflows

## Example
```
/phase5-test
```
