# Setup Local Development

Setup local development environment for the Todo AI Chatbot.

## Usage
```
/dev-setup
```

## What it does
1. Checks and installs required dependencies
2. Sets up local Kubernetes (Minikube/Kind)
3. Configures local Docker registry
4. Creates development namespace
5. Deploys development services (database, Redis, etc.)
6. Sets up hot-reload for development
7. Configures environment variables

## Example
```
/dev-setup
```
