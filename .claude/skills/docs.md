# Generate Documentation

Generate comprehensive documentation for the Todo AI Chatbot project.

## Usage
```
/docs [type]
```

## Arguments
- `type` (optional): Documentation type (api/architecture/deployment/user/all). Default: all

## What it does
1. **api**: Generates API documentation from code
2. **architecture**: Creates architecture diagrams and docs
3. **deployment**: Documents deployment procedures
4. **user**: Creates user guides and tutorials
5. **all**: Generates complete documentation set

## Example
```
/docs
/docs api
/docs architecture
```
